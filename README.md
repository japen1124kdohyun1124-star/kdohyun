# 📚 StudySub - 공부 웹사이트

학년과 과목별로 공부 문제를 검색하고 풀 수 있는 웹사이트입니다.

## 🚀 시작하기

### 요구사항
- Node.js 14 이상
- npm 또는 yarn

### 설치

```bash
# 프로젝트 클론
git clone https://github.com/japen1124kdohyun1124-star/kdohyun.git
cd kdohyun

# 의존성 설치
npm install
```

### 실행

```bash
# 개발 서버 시작
npm start

# 브라우저에서 http://localhost:3000 열기
```

### 배포

```bash
# GitHub Pages에 배포
npm run deploy
```

## ✨ 주요 기능

- ✅ 학년 선택 - 초등학교부터 고등학교까지
- ✅ 과목 선택 - 국어, 수학, 영어, 과학, 사회, 역사
- ✅ 질문 필터링 - 선택한 학년과 과목에 맞는 문제 조회
- ✅ 난이도 표시 - 쉬움, 보통, 어려움
- ✅ 반응형 디자인 - 모바일/태블릿/데스크톱 지원
